⌂ Головна ⚙ Налаштування Допомога

Для роботи із програмою, вам потрібно зробити наступні налаштування:

  • TELEGRAM_PHONE_NUM - телефон вашого акаунту в Телеграм
  • TELEGRAM_API_ID - айді вашого застосунку в Телеграм
  • TELEGRAM_API_HASH - хеш вашого застосунку в Телеграм
  • TELEGRAM_ITERATION_DELAY - пауза в секундах між циклами постінгу (потрібно для уникнення блокування)
  • TELEGRAM_API_DELAY - пауза в секундах між викликами API (потрібно для уникнення блокування)

Як отримати Telegram API ID та API Hash

  1. Перейдіть на сайт Telegram для розробників
    https://my.telegram.org
  2. Увійдіть за номером телефону
    • Введіть номер у міжнародному форматі (наприклад, +380...)
    • Отримайте код підтвердження у Telegram
    • Підтвердіть вхід
  3. Виберіть пункт API development tools
  4. Заповніть форму створення застосунку
    • App title — будь-яка назва (наприклад, My App)
    • Short name — коротка назва латиницею
    • Platform — Desktop або Other
    • URL — можна залишити порожнім
  5. Після збереження Ви отримаєте:
    • api_id — числовий ідентифікатор
    • api_hash — секретний ключ

Для роботи програми, двофакторна автентифікація (2FA) у вашому акаунті має бути вимкнена:

Для активації штучного інтелекту для реплаїв, виконайте наступні дії:

1️⃣ Реєстрація в GroqCloud

  1. Перейдіть на сайт GroqCloud: https://console.groq.com
  2. Натисніть Sign Up або Log In
  3. Увійдіть за допомогою Google або email

Після входу Ви потрапите в Groq Console.

2️⃣ Отримання GROQ_API_KEY

  1. Відкрийте розділ API Keys
  2. Натисніть Create API Key
  3. Скопіюйте згенерований ключ, наприклад: 
    gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxx

3️⃣ Збереження GROQ_API_KEY в налаштуваннях:

  1. Створіть параметр конфігурації із назвою "GROQ_API_KEY" та збережіть у нього згенерований ключ

Інші налаштування:

  • TELEGRAM_POLL_INTERVAL - пауза між опитуванням каналів про нові пости
  • TELEGRAM_NEW_POST_DEBOUNCE - пауза перед публікацією коментаря під новим постом
  • TELEGRAM_NEW_POST_MAX_AGE - максимальний вік нового поста
  • TELEGRAM_SEND_AS_CHANNEL - канал від імені якого надсилати повідомлення
  • TELEGRAM_RESTORE_BIO_DELAY - пауза перед відновленням інформації в профілі
  • TELEGRAM_EDIT_DELAY - пауза перед редагуванням повідомлення
  • TELEGRAM_DELETE_DELAY - пауза перед видаленням повідомлення
